Sitio Web Corporativo para Capitalprime

Desarrollé de forma integral el sitio web corporativo de Capitalprime, encargándome del frontend, backend y la base de datos. La web es completamente responsiva, dinámica y parametrizable. Presenta la información institucional de la empresa e integra catálogos dinámicos de servicios, además de formularios interactivos para contacto y solicitudes de cotización.

Digital Waiter: Aplicación movil para restaurantes.

Desarrollé una aplicación movil integral para restaurantes que combina funcionalidades informativas y de interacción con los clientes. La aplicación permite a los comensales explorar el menú digital y realizar pedidos, desarrollado en React Ntive. El API que consume la aplicación está desarrollado en Laravel

Digital Waiter: API de menus y pedidos.

API para la aplicación Digital Waiter en Laravel, este API se comunica con una base de datos en PostgreSQL de donde obtiene todos los parámetros de los menus y permite guardar los pedidos.
